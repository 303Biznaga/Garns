# GARNS – Sudoku de Howard Garns

Sudoku progresivo con autoguardado, sonidos a 432 Hz y PWA para jugar sin conexión.

## 🎮 Características
- 4 dificultades (Fácil, Medio, Difícil, Experto)
- Autoguardado de partida (localStorage)
- Sonidos sintetizados en tiempo real (432 Hz)
- Atmósfera de fondo sutil
- Instalable como app (PWA)
- Modo notas y pistas

## 🚀 Cómo jugar
1. Elige una dificultad
2. Toca una celda vacía y luego un número
3. Usa el botón 📝 para activar el modo notas
4. Completa filas, columnas y cuadrantes para sumar puntos y combo

## 🛠️ Tecnologías
- HTML5, CSS3, JavaScript
- Web Audio API (síntesis de sonidos)
- Service Worker (PWA)
- LocalStorage (autoguardado)
